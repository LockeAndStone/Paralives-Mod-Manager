# Paralives-Mod-Manager
Mod manager for Paralives

## Current Features
- Tracks both Local and Workshop Mods
- Allows conversion of Workshop to local mods (Must manually un-subscribe from the workshop mod after)
- You can add mods from zip files by button or by simply drag and dropping them anywhere on the Mod Manager
- You can enable/disable mods by simply checking/unchecking a checkbox next to the mod in the list (Make sure to press save changes!)
- You can filter mods by searching in the search bar. Realtime searching! (Press the 'esc' key to clear the search instantly)
- A refresh button is included so that if you subscribe to mods or manually install mods, you can force and update to the list.
- You can delete local mods only. Workshop mods should just be unsubscribed from steam, unless you have converted them already.
- The ability to Launch the game directly from the mod manager. (It will prompt to save changes if you havent already done so.)
- You can still use the ingame mod manager to disable/enable mods, and these changes will be reflected in the mod manager by pressing the refresh button.
